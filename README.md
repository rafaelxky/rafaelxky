## ML From Scratch
An implementation of a predictive model built from scratch.

It uses gradient descent and a basic neural network structure, implemented entirely with mathematical primitives. 
I tested different activation functions to observe their effect on convergence and behavior.

The project also served as an entry point into GPU programming. 
The results are simple by modern standards, but the implementation helped clarify how learning systems are constructed from first principles.
You can call it "Artificial Inteligence" but it sure is dumb

## RPlace
A macro-based domain-specific language for generating structured text.

It supports:
- Template composition
- Conditional insertion
- Imports
- Inheritance
- Overloading

The goal was to reduce repetitive scaffolding in text-heavy projects. 
The design evolved gradually as the implementation revealed constraints and edge cases.
Making a programing language was something unique for me.
They used to be abstract pieces of software that allowed me to make projects but It gives me a new light to undestand how they work

## Java Blockchain
